# Gemini API

Ejemplo de conexión a Gemini con Python.

## Requisitos

- Tener definida la variable de entorno `GEMINI_API_KEY`.

## Ejecutar

1. Activa el entorno virtual.
2. Instala dependencias con `pip install -r requirements.txt`.
3. Ejecuta `python app_gemini.py`.
